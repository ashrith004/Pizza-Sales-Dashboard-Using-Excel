🍕 Pizza Sales Dashboard

An interactive sales analytics dashboard built using Excel and MySQL to analyze pizza store performance, customer ordering patterns, and revenue trends.
The project focuses on transforming raw order data into actionable business insights.

📊 Project Overview

This dashboard analyzes 21,000+ pizza orders generating $8.17M in revenue to understand sales trends, customer behavior, and product performance.

The analysis helps answer key business questions such as:

When are peak sales hours?

Which pizzas generate the most revenue?

Which products underperform?

What days generate the highest orders?

🔍 Key Insights
⏰ Peak Sales Hours

Highest orders occur between 12–1 PM and 5–8 PM

Lunch and dinner drive the majority of revenue

📅 Sales by Day

Weekends generate the highest sales

Saturday shows the highest order volume

🏆 Top Performing Pizzas

The Top 5 pizzas contribute the majority of revenue

Large-sized pizzas generate the highest sales

📉 Lowest Performing Pizzas

Bottom 5 pizzas contribute minimal revenue

These products can be reviewed for menu optimization

📊 Dashboard Features

The interactive dashboard includes:

Revenue by pizza category

Orders by day of the week

Sales by pizza size

Top 5 and bottom 5 pizzas

Hourly sales trends

Dynamic filters using slicers
